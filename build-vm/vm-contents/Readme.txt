1.Click the "Select Files" icon in the upper right corner of the Refactoring View (looks like a normal folder icon) shown in the bottom alongside the Console Window

2.Select from the dropdowns the before and after project versions from your workspace and press ok !

3.A progress dialog will appear (but may not share the most useful data). Comparison and refactoring inference should take less than ten minutes; then you will see the output.

4.On the left you will see a list of found refactorings. Double click to pull up more information on the right.

5.Double click an old or new code file to pull up the corresponding file
